2. 10개 사업 카테고리 구조

각 사업 카테고리는 독립된 현황 데이터를 가지며, 관리자 모드에서 개별 입력/수정/삭제가 가능합니다.

필드값 예시

필드값 예시

③ KCMVP (암호모듈검증)

필드값 예시
